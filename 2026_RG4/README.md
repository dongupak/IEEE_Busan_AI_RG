## 2026년 AI 연구회 4차 특강
* 일시 :  2026년 2월 25일 수요일 오후 3:00 - 5:50
* 장소 : 동서대학교 뉴밀레니엄관(13번, 본관) 3층 333호
* 강사 :  동서대학교 소프트웨어학과 이상걸 교수
* 강연 제목 : 생성형 AI 오케스트레이션: ComfyUI와 오픈소스 모델을 활용한 콘텐츠 생성 실습
* 특강 소개글 :
상상이 현실로 구현되는 생성형 AI의 시대, 이제 개별 모델을 사용하는 단계를 넘어 여러 AI를 조화롭게 지휘하는 '오케스트레이션' 능력이 핵심 경쟁력이 되었습니다. 텍스트가 이미지가 되고, 이미지가 다시 생동감 넘치는 비디오와 음악으로 이어지는 창작의 흐름은 콘텐츠 제작의 패러다임을 완전히 바꾸고 있습니다.
이번 특강에서는 노드 기반의 비주얼 프로그래밍 도구인 ComfyUI를 활용하여 최신 오픈소스 AI 모델들을 체계적으로 다룹니다. 단순히 툴을 익히는 것을 넘어, 고해상도 이미지 생성(Flux 등)부터 고품질 비디오 제작(CogVideoX & HunyuanVideo), 그리고 전문적인 음악 생성(Stable Audio 등)까지 아우르는 전체 파이프라인을 직접 설계하고 실습합니다.
양자화 모델을 통한 하드웨어 최적화 기법과 멀티모달 콘텐츠를 하나의 워크플로우로 통합하는 실전 노하우를 확인할 수 있습니다. 인공지능 연구, 차세대 콘텐츠 제작, 그리고 에이전틱(Agentic) 워크플로우 설계에 관심 있는 교수님과 연구원분들께 실질적인 기술적 영감을 제공할 이번 특강에 여러분을 초대합니다.
